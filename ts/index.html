Правила

  1. Предотвратить переопределение переменных (no-redeclare)

  3. Использовать var в начале функции (vars-on-top)
  4. Предотвратить переопределение переменных из замыкания выше (no-shadow)

ES2015

let и const

Отличия от var:

  1. Область видимости
  2. Не возможно объявить повторно
  3. Отсутствие "hoisting"

Область видимости

				justDoIt = function () { 
					if (true) { 
						var a = 9; 
					}
				}

Область видимости

				justDoIt = function () { 
					if (true) { 
						let a = 9; 
					}
				}

Не возможность объявить повторно

				justDoIt = function () { 
					var a = 9;
					var a = 3; 
				}

Почему Flow?

  1. Flow осуществляе статический анализ
  2. Синтаксис очень похож на TypeScript
  3. Flow все типы non-nullable
  4. Поддержка TypeScript библиотек типов
