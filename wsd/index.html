NaCl и PNaCl

Почему NaCl и PNaCl?

  1. Можно использовать написанный С/С++ код
  2. Супер быстрый (почти как C++)
  3. Безопасность

Что нужно сделать:

  1. Скомпилировать С/С++
  2. Создать манифест файл
  3. Добавить embed элемент с ссылкой на манифест
  4. Готово!

Как писать на asmjs?

  1. Пишем на C++
  2. LLVM (Clang)
  3. Emscripten
  4. JavaScript

Что может C/С++?

  1. Работа с файловой системой -> FS и indexDB (embedded)
  2. Работа с "кучей" для выделение памяти -> Int32Array
  3. Библиотеки для работы с графикой -> WebGL
  4. Потоки -> WebWorkers

Что не может asmjs?

  1. Низкоуровневые операции
  2. Работа с числами
  3. Общая пямять для потоков (SharedBufferArray)
